- 提升技术人员的表达能力
## 二、学习形式
- 时间：每周三晚上6:30 至 7:30
- 方式：
每周有一人分享一个知识点，其他人提前预习，分享后进入讨论环节
- 要求：
  > 知识点讲解 
  
  > demo演示与讲解
  
  > 应用场景说明
